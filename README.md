# 1.Hysteria2在Nodejs或Python部署

* Node.js/Python运行环境一键极简部署Hysteria2节点，可自定义端口

* 更新自适应端口，无需再手动设置

```
curl -Ls https://raw.githubusercontent.com/eishare/tuic-hy2-node.js-python/main/hy2.sh | sed 's/\r$//' | bash
```
